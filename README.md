# Alke Wallet

Alke Wallet es un simulador de billetera virtual que permite a los usuarios gestionar su dinero de forma digital. Fue mi primer proyecto de desarrollo web, diseñado para consolidar conocimientos en lógica de programación y persistencia de datos en el cliente.

## 🚀 Funcionalidades Principales

* **Autenticación de Usuario:** Sistema de login simulado para proteger el acceso a la billetera.
* **Gestión de Saldo:** Visualización y actualización dinámica del saldo disponible en tiempo real.
* **Operaciones Financieras:** Módulo funcional para realizar depósitos y transferencias entre contactos.
* **Historial de Movimientos:** Registro detallado de cada transacción, incluyendo descripción, monto y marca de tiempo.
* **Persistencia de Datos:** Implementación de `localStorage` para guardar el estado de la cuenta y el historial sin base de datos externa.

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3:** Estructura semántica y diseño personalizado con gradientes avanzados.
* **Bootstrap 5:** Framework de UI para garantizar un diseño responsivo y profesional de forma ágil.
* **JavaScript (Vanilla):** Gestión de la lógica financiera y cálculos de saldo.
* **jQuery:** Simplificación de la manipulación del DOM y manejo de eventos del formulario.
* **GitHub Actions:** Automatización para el despliegue continuo (CI/CD) a través de GitHub Pages.

## 🔧 Detalles de Implementación

Para este proyecto, me enfoqué en:
1.  **Validación de entradas:** Controlar que los montos ingresados sean válidos para evitar errores en el saldo.
2.  **Redireccionamiento Inteligente:** Lógica de control que verifica si el usuario está autenticado antes de mostrar el menú principal.
3.  **Inyección de contenido:** Generación dinámica de la lista de transacciones en el historial utilizando arreglos de objetos y bucles.
